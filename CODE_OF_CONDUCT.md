# Contributor Covenant Code of Conduct

## Our Pledge
We pledge to foster an open, welcoming, inclusive, and safe environment. This project is educational and synthetic; kindness and respect are required.

## Our Standards
- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Give and gracefully accept constructive feedback
- Focus on what is best for the community

## Our Responsibilities
Maintainers may remove or reject contributions that violate this Code of Conduct.

## Scope
This Code applies within project spaces and in public spaces when representing the project.

## Enforcement
Report issues via the repository issue tracker. Maintainers will review and act as appropriate.
